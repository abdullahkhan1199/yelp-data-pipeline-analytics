# Yelp Data Pipeline Analytics

## Overview
This project uses Yelp data to analyze restaurant and customer trends across multiple U.S. cities. The work involved transforming semi-structured Yelp data into a structured format, running SQL-based analysis, and presenting business insights through dashboards and presentations.

## Objective
The goal of this project was to use Yelp data to support data-driven restaurant planning decisions, including where to launch, how to price, when to operate, and which service model to prioritize.

## Methods Used
- SQL Querying
- Data Cleaning and Transformation
- Azure SQL
- KNIME Workflows
- Tableau / Data Visualization
